100 valuable blogs for developers

March 9, 2024

Comprehensive collection of tech blogs to explore the technological and development landscape.

#Languages and standards

  • ECMAScript Daily – ECMAScript proposal updates and news.
  • Microsoft TypeScript Blog – Updates and insights from the TypeScript team at Microsoft.
  • WHATWG Blog – News and updates from the Web Hypertext Application Technology Working Group.
  • ReScript Blog – Updates, releases, and feature announcements for the ReScript programming language.
  • Dart Blog – Announcements and development updates for the Dart programming language.
  • Rust Blog – Official blog for the Rust programming language, including release notes and development updates.
  • Python Blog – Core development news and information for the Python programming language, including new features and updates.
  • Kotlin Blog – Updates and news from the Kotlin team at JetBrains, including tutorials and community highlights.
  • Scala Blog – News, updates, and community highlights from the Scala programming language community.

  • SpiderMonkey Blog – News and updates from the SpiderMonkey team, the JavaScript engine behind Firefox and other applications.

  • Caniuse News – Aggregated news and updates on web technology support across different browsers, including new features and deprecations.
