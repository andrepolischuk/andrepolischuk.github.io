Valuable blogs for frontend developers

March 9, 2024

Comprehensive collection of tech blogs to explore the technological and development landscape.

#Languages and standards

  • ECMAScript Daily – ECMAScript proposal updates and news.
  • Microsoft TypeScript Blog – Updates and insights from the TypeScript team at Microsoft.
  • WHATWG Blog – News and updates from the Web Hypertext Application Technology Working Group.

#Browsers and platforms

  • Browsers.fyi – Platform that tracks the latest browser versions and their release notes, offering an API for developers and an RSS feed for updates.
  • Chromium Blog – Official blog for the Chromium project, covering development updates, new features, and releases.
  • Google Chrome Blog – News, updates, and insights from the Google Chrome team, including new features and security improvements.
  • Chrome Developers Blog – Collection of posts from the Chrome team on web development, tools, and updates.
  • WebKit Blog – Updates and news from the WebKit team, including new features and improvements in the WebKit rendering engine.
  • V8 Blog – News and updates from the V8 team, focusing on the V8 JavaScript engine and its improvements.
  • Mozilla Future Releases Blog – Announcements and updates on upcoming releases and features for Mozilla products.
  • Mozilla Firefox Blog – News and updates from the Firefox team, including new features, security updates, and development insights.
  • Mozilla Nightly Blog – Insights and updates from the team behind Mozilla Nightly, a pre-release version of Firefox.
  • Mozilla Nightly Notes – Release notes for the latest version of Mozilla Nightly, detailing new features and bug fixes.
  • SpiderMonkey Blog – News and updates from the SpiderMonkey team, the JavaScript engine behind Firefox and other applications.
  • Servo Blog – Updates and news from the Servo project, an experimental web browser engine.
  • Microsoft Edge Developer Blog – Insights and updates from the Microsoft Edge team, focusing on web development and the browser’s features.
  • Caniuse News – Aggregated news and updates on web technology support across different browsers, including new features and deprecations.

#Frameworks and technologies

  • Astro Blog – Latest news, resources, and documentation for Astro.
  • Angular Blog – News and tips from the Angular Team, covering updates, features, and announcements.
  • ESLint Blog – The latest ESLint news, case studies, tutorials, and resources, including updates on releases, new features, and bug fixes.
  • Ionic Blog – Insights into Ionic, focusing on products like Appflow, Capacitor, Enterprise SDK, and more.
  • Cypress Blog – Updates, best practices, and tutorials on using Cypress for end-to-end testing of web applications.
  • Remix Blog – Featured articles on open development, integrating Remix with Shopify, React Router, and more.
  • Vite Blog – Latest updates, announcements, and discussions about Vite.
  • Prettier Blog – Recent updates on Prettier, an opinionated code formatter.
  • Apollo GraphQL Blog – Latest updates, best practices, and community insights on Apollo Client, Apollo Federation, GraphQL, and more.
  • Yarn Blog – Recent releases, and updates on Yarn.
  • Bun Blog – Collection of updates and release notes for Bun, a new JavaScript runtime with a bundler, transpiler, task runner, and more.
  • Deno Blog – Updates on Deno releases, including product updates, engineering insights, and how-to guides for various use cases.
  • Svelte Blog – Updates on Svelte and SvelteKit, including new features, improvements, and community news.
  • React Blog – The official source for updates and release notes from the React team.
  • React Native Blog – Updates on React Native releases, features, and community news.
  • Vue.js Blog – Latest news, updates, and features of Vue.js, including releases, improvements, and community discussions.
  • Electron Blog – Insights, updates, and tutorials on Electron, a framework for building cross-platform desktop apps with web technologies.
  • Node.js Blog – Official blog for Node.js, featuring updates on new releases, along with security releases and patches.
  • Next.js Blog – Features, updates, and best practices for building modern web applications with Next.js.
  • Lit Blog – Latest updates, release announcements, and showcases for Lit.
  • Babel Blog – Latest updates, features, and releases of Babel.
  • Docker Blog – Platform for sharing Docker’s latest news, updates, and insights related to containerization, Docker products, and industry trends.
  • Flutter Blog – Latest updates, release notes, and announcements of Flutter, a framework for crafting high-quality native interfaces on iOS, Android, web, and desktop.

#Community

  • web.dev Blog – Collection of posts covering web development best practices, updates on web standards, and tools for building better websites.
  • Frontend Almanac Blog – Collection of articles and resources dedicated to frontend development, covering topics from CSS to JavaScript, accessibility, and performance.
  • Web Platform News – Latest news, updates, and insights related to web development and web technologies, including accessibility, CSS, JavaScript, and more.

#Personal

  • Hemanth.hm on DEV – Updates from various TC39 meetings.
  • Addy Osmani’s Blog – Collection of articles focused on engineering, development practices, and insights from Addy Osmani, an Engineering Manager at Google working on Chrome.
  • Industrial Empathy – Personal blog by Malte Ubl, focusing on large-scale software design and web development, with insights into web development and performance, software design and architecture, and personal news.

#Security and privacy

  • eSecurity Planet – News, insights, and resources on security, compliance, and technology.
  • Google Security Blog – The official Google Security Blog, sharing news and updates on security, privacy, and data protection.
  • Mozilla Security Blog – Updates on Mozilla’s security efforts, including Firefox enhancements, VPN security audits, and discussions on privacy and security policies.
  • Microsoft Edge VR – Insights, updates, and stories related to Microsoft Edge’s security, development, and community efforts, including enhanced security features, and vulnerability research.
  • Privacy Sandbox – Collection of resources, tools, and discussions aimed at advancing privacy-preserving technologies and practices on the web.

#ML and AI

  • Unite.AI – AI news, surveillance, AI bias, robotics, AGI, disruptive technologies.
  • Ahead of AI – Specialized magazine in Machine Learning and AI research, read by a wide audience of researchers and practitioners looking to stay ahead in the evolving field of AI.
  • Data Machina – Newsletter that provides insights into the world of data science, machine learning, and AI.

#Bonus

  • Hacker News Best – Collection of the most upvoted and discussed posts on Hacker News, covering a wide range of topics from technology, programming, and startups to science, philosophy, and more.
  • Quanta Magazine – Science, technology, and the future of the world, with a focus on providing in-depth reporting and analysis on the latest discoveries and developments.
